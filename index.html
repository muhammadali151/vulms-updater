Help Center

📥 How do I install the app?

1. Tap Download & Update App.
2. Wait for the APK to finish downloading.
3. Open the downloaded file.
4. If Android blocks the installation, allow Install Unknown Apps for your browser or file manager.
5. Tap Install.

🔄 How do I update the app?

Download the latest APK from this page and install it over your existing app. Your saved data and settings will remain intact.

❓ The APK won't install. What should I do?

• Make sure you downloaded the complete APK.
• Enable Install Unknown Apps in Android settings.
• Ensure you have enough storage space.
• If the problem continues, uninstall any conflicting version before installing again.

🔐 Will my student account be safe?

Yes. Your saved account information is stored locally on your own device. The app does not upload your credentials to any external server.
